# Arquivos de Teste

Este repositório contém arquivos de teste para diversos propósitos de verificação e validação de código.

## Descrição dos Arquivos de Teste

- **teste1.js**: Testa a funcionalidade básica de entrada e saída de dados.
- **teste2.js**: Verifica a lógica de cálculo de valores com diferentes parâmetros.
- **teste3.js**: Simula casos de uso para tratamento de erros e validação de dados.
- **teste-ui.html**: Contém testes de interface de usuário para verificar responsividade e elementos visuais.
